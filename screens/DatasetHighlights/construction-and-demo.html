Construction and Demolition Permits and Code Violations Included in Landmark WPRDC Project Parcels N'At

Data published on construction and demolition permits and code violations from the OneStop system are used by WPRDC in the exemplary project Parcels N’At, a data explorer and bulk download tool that provides data about parcels in Allegheny County.

Parcels N’At displays data collected at the individual parcel level from Allegheny County data sources for property assessments, filed assessment appeals, historical assessment appeals, property sales transactions, mortgage foreclosure records, tax liens, conservatorship filings, and for City properties, permits and code violations.

The data can be visualized in the data explorer for any parcel in Allegheny County. City parcels have the additional feature of permits and code violations. Each of the data elements are downloadable in a dataset for individual or a set of parcels, neighborhoods, and municipalities in the bulk download tool.

Data elements displayed for construction and demolition permits include Permit ID, Issue Date, Permit Type, Work Type, Total Project Value, Contractor Name, and Work Description.

Data elements displayed for code violations include the investigation history (findings and outcome), closed/ open status, violation, date opened, last investigation, code section, and instructions to remediate. Investigation history displays.

All are available for dataset download and via CKAN’s Datastore OData API at the links above.
